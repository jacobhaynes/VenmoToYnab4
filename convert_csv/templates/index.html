Convert Venmo to YNAB4

CONVERSTION

  • Open the Venmo statement page:
    https://venmo.com/account/statement
  • Select a date range.
  • Click "Download CSV".
  • Select file below.
  • Press submit and download. The converted file should start downloading.
  • Open in YNAB 4 with "File"->"Import Transactions from Your Bank".
